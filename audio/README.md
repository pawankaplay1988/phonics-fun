## Adding Phonics Audio Files

The app expects one MP3 file per letter in this folder:

```
audio/a.mp3
audio/b.mp3
audio/c.mp3
...
audio/z.mp3
```

### Requirements
- Each file should contain ONLY the phoneme sound (not the letter name)
- Keep files short: 1–2 seconds
- Clear, isolated phoneme pronunciation
- Recorded by a native English speaker (ideally female, soft voice for kids)

### Free Sources for Phonics Audio

1. **Record your own** — Use your phone or computer microphone
2. **Jolly Phonics** style — Search for "letter sound" clips (ensure license allows use)
3. **Oxford Phonics World** — Reference for correct sounds
4. **Forvo.com** — Community-pronounced sounds (check licenses)

### If no audio files are present
The app will display a friendly message: "Phonics audio not available" and still show all example words with speech synthesis for word pronunciation.
